# Platform names

The Bowie namespace — a reference sheet, nothing more. What each name is attached to lives in `FEATURES.md`; nothing here implies a platform exists, is scoped, or is being built.

Codenames are single-word, and should read as a name rather than a noun — generic dictionary words collide with platform vocabulary in running prose and don't announce themselves as codenames.
