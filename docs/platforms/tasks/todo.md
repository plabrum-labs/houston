# Tasks — todo

A task / issue-tracking primitive (Linear-like) an app can pull in.

- Source: Houston (net-new)

## What it is

A first-class task / issue primitive — items, status, assignment, ordering — that a vertical app
composes in rather than re-deriving, modeled on Linear.
